## 2. Brand Voice & Tone

### Voice Principles

**1. Calm Competence**
love.Me never panics, never apologizes excessively, never hedges. It states what happened and what it's doing about it.

```
No:  "Oops! Something went wrong. We're so sorry!"
No:  "Uh oh! The connection was lost."
Yes: "Connection lost. Reconnecting now."
Yes: "That tool returned an error. Trying a different approach."
```

**2. Warm Brevity**
Every word earns its place. Warmth comes from directness and respect for the user's time, not from filler words or emoji.

```
No:  "Hey there! ðŸ‘‹ Welcome to love.Me! We're super excited to have you!"
No:  "Welcome. Please configure your daemon connection parameters."
Yes: "Welcome to love.Me. Let's connect to your Mac."
```

**3. First Person Singular**
love.Me speaks as "I" â€” it's a personal relationship, not a corporate product.

```
No:  "We encountered an error."
No:  "The system is processing your request."
Yes: "I ran into an error with that file."
Yes: "Working on it now."
```

**4. Show, Don't Reassure**
Instead of saying "don't worry," show the work. The thinking panel and tool feed ARE the reassurance.

```
No:  "Don't worry, I'm working on it!"
Yes: [Thinking panel streams reasoning] [Tool cards show file reads]
```

### Tone Spectrum

| Context | Tone | Example |
|---------|------|---------|
| Onboarding | Warm, inviting | "Let's connect to your Mac." |
| Chat (idle) | Ready, attentive | "Message love.Me..." |
| Working | Focused, transparent | Thinking... â†’ tool calls â†’ response |
| Success | Understated confidence | "Done. Deployed to staging." |
| Error | Calm, solution-oriented | "Connection lost. Reconnecting." |
| Settings | Clear, instructional | "Your Mac's local IP address." |

### Words We Use

| Instead of... | We say... |
|--------------|-----------|
| Error occurred | Ran into an issue |
| Processing | Working on it |
| Invalid | Doesn't look right |
| Retry | Try again |
| Configure | Set up |
| Terminate | Stop |
| Execute | Run |
| Successful | Done |

### Words We Never Use
- "Oops," "Uh oh," "Whoops"
- "Super," "Amazing," "Awesome"
- "Please wait" (just show the work happening)
- "AI-powered" (we ARE the AI, no need to label it)
- "Smart" or "Intelligent" (show it, don't claim it)
- Corporate jargon: "leverage," "utilize," "facilitate"

## 6. Brand Guidelines Summary

### Logo Usage
- `love` lowercase, light weight
- `.` in Heart color (#E8456B), slightly oversized
- `Me` capitalized, semibold weight
- Minimum clear space: 1x height of "M" on all sides
- Never stretch, rotate, or recolor
- On dark backgrounds: white text + Heart dot
- On light backgrounds: Deep Ink text + Heart dot

### Color Rules
- Heart (#E8456B) is used ONCE per screen as the primary action
- Never use Heart for errors (use Soft Red #EF4444)
- Never use Heart for backgrounds (except user bubbles)
- Deep Ink (#1A1A2E) is the default canvas
- Trust (#6B7B8D) for anything secondary or muted

### Imagery
- No stock photography
- No illustrations or mascots
- No gradients (flat colors only)
- Interface screenshots are the primary visual asset
- The product IS the brand â€” let the UI speak

### Tagline Usage
- Primary: "Your AI, working for you."
- Short: "Your AI."
- Technical: "AI agent for your machine."
- Never use without the product name nearby
